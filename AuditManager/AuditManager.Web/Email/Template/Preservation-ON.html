What you need to know:
  • As noted above, nothing in this workspace should be deleted or altered. If necessary, you may make a copy of information contained within the workspace. If you need to change anything in this workspace, you will need to create a new version and work on and change that version only. Utilize the ‘copy as new version’ function to do so.
  • You should copy, rather than move, emails from your PSTs to this workspace by using the Outlook Edit menu and choosing the ‘Copy to Folder’ option.
  • If you had already checked out documents from this workspace, you must check them back in as a new version, and preserve any copies outside of the workspace (e.g. copies on your computer). Make sure the “Delete local file after check in” box is un-checked when you follow the check-in instructions.
  • As a reminder, if you previously printed any documents and/or records from this workspace, you must preserve those hard-copy versions as well. New folders and documents can still be created in this workspace, unless you have been otherwise advised by OGC.
  • You will be able see the preservation ‘flag’ by viewing the status field within the properties of the DRMS workspace and documents.
